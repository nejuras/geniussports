# Genius Sports

# Requirements
```
 - PHP 7.0+
 - Mysql 5.7.x
```

# How to Run dev environment
```
 $ git clone <project>
 $ cd path/to/<project>
 $ composer install
 $ php bin/console doctrine:database:create
 $ php bin/console doctrine:schema:update --force
 $ php bin/console npm install
 $ php bin/console server:run

```
